# Документация проекта «Шаверма БН»

Документация фиксирует продуктовую концепцию, состав MVP, архитектуру и порядок развития сервиса предзаказа.

## Документы

- [MVP](./MVP.md) — что именно строим в первой рабочей версии.
- [Архитектура](./ARCHITECTURE.md) — компоненты системы и их ответственность.
- [Продуктовые принципы](./PRINCIPLES.md) — ключевые ограничения и подходы.
- [Флоу заказа](./ORDER_FLOW.md) — клиентский и Telegram-сценарии.
- [Дорожная карта](./ROADMAP.md) — этапы развития до оплаты по СБП.
- [Зафиксированные решения](./DECISIONS.md) — журнал принятых решений.

## Текущий статус

Работает клиентская PWA-витрина с актуальным меню:

- категории;
- карточки товаров;
- допы;
- корзина;
- выбор времени;
- экран номера заказа;
- сохранение корзины и последнего заказа в браузере.

Пока заказ существует только в браузере пользователя. Следующий этап — backend, база заказов и Telegram-бот.
