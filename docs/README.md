Этот раздел фиксирует текущую продуктовую концепцию, архитектуру и порядок развития сервиса предзаказа.

## Текущий статус

На текущем этапе работает статическая PWA-витрина:

- опубликована через GitHub Pages;
- содержит актуальное меню и цены;
- позволяет выбирать товары и допы;
- формирует корзину;
- позволяет выбрать время получения;
- показывает номер заказа;
- хранит корзину и последний заказ только в браузере пользователя.

Заказы пока не отправляются на точку. Номер заказа генерируется локально и используется только для демонстрации клиентского сценария.

Следующий крупный этап — подключение серверной части, базы заказов и Telegram-бота.
