# Шаверма БН — PWA-прототип

Версия витрины, пересобранная под актуальное меню заведения.

## В меню

- шаверма;
- мангал;
- хот-доги;
- картофель, соусы;
- горячие напитки;
- допы к шаверме и хот-догам.

## Сейчас работает

- адаптивная мобильная витрина;
- категории;
- реальные позиции и цены из текущего меню;
- выбор допов;
- корзина с количеством;
- выбор времени получения;
- анонимное оформление;
- номер заказа;
- сохранение корзины и последнего заказа в браузере;
- установка как PWA;
- GitHub Pages без дополнительной сборки.

## Важно

Пока это автономный фронтенд-прототип:

- заказ никуда не отправляется;
- номер генерируется локально;
- меню зашито в `app.js`;
- Telegram и сервер ещё не подключены.

## Публикация на GitHub Pages

1. Распакуйте архив.
2. Загрузите все файлы в корень репозитория.
3. Откройте `Settings → Pages`.
4. Выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/ (root)`.

## Локальный запуск

```bash
python -m http.server 8080
```

Откройте:

```text
http://localhost:8080
```
